# Python-scripts-learner
Foundational study projects in Python. A collection of python scripts I am writing as  i learn the fundamentals of programming

## Projects Included

### 1. Compound Interest Calculator
A simple tool that calculates the future value of an investment based on user input.
* Key features: Uses 'while' loops for input validation and 'pow()' function for calculations.
### 2. Countdown Timer
A simple script that takes user input in seconds and counts down in an HH:MM:SS format
* Key features: Uses time.sleep(1) for the delay, modulud operator(%) and integer division to format the time.
### 3. Shoping cart program
A simple python program that takes user input and places it in an empty list.
* Key features: Uses lists to store data, while loop to collect data, for loop to print and calculate.
